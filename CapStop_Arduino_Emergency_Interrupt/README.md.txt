# CapStop: Arduino Emergency Interrupt

## Introdução
CapStop é um sistema de interrupção de emergência para Arduino, utilizando sensores capacitivos. Este projeto visa aumentar a segurança em aplicações que necessitam de um mecanismo de parada imediata.

## Lista de Materiais Necessários
- Arduino (qualquer modelo compatível)
- 2x Sensores Capacitivos
- Cabos de conexão
- Resistores (opcional, dependendo do sensor)

## Propósito do Projeto
Este projeto foi criado para oferecer uma solução simples e eficaz para interromper qualquer operação do Arduino em casos de emergência, como em sistemas robóticos ou automatizados onde a intervenção humana rápida é crucial.

## Prós e Contras
### Prós:
- Fácil de implementar e integrar com outros projetos.
- Aumenta significativamente a segurança.

### Contras:
- Dependente da sensibilidade e calibração dos sensores capacitivos.

## Orientações Gerais
### Configuração:
1. Conecte os sensores capacitivos aos pinos especificados no código.
2. Carregue o código no Arduino.

### Uso:
- O sistema entra em ação quando um dos sensores capacitivos detecta um objeto próximo.

### Solução de Problemas:
- Garanta que os sensores estejam corretamente calibrados e livres de interferências.
