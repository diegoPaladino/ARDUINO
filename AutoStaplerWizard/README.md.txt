# AutoStaplerWizard

## Introdução
AutoStaplerWizard é um projeto Arduino que visa automatizar um grampeador de papel manual. Utiliza um sensor de luz para detectar a presença de papel e um servo motor para acionar o grampeamento.

## Lista de Materiais Necessários
- Arduino UNO
- Servo motor
- Sensor de luz (LDR)
- Resistores
- Fios de conexão
- Grampeador modelo G105 ou GE1037

## Propósito do Projeto
Este projeto foi criado para automatizar o processo de grampeamento de papéis, tornando-o mais eficiente e menos trabalhoso. É ideal para escritórios ou ambientes educacionais onde o grampeamento frequente de documentos é necessário.

## Prós e Contras
**Prós:**
- Automatiza uma tarefa repetitiva.
- Pode ser customizado para diferentes modelos de grampeadores.

**Contras:**
- Requer montagem e configuração iniciais.
- Limitado ao tamanho e tipo de papel que o grampeador manual pode aceitar.

## Orientações Gerais
1. Conecte o servo motor ao pino 9 do Arduino.
2. Conecte o sensor de luz ao pino A0.
3. Carregue o código no Arduino.
4. Ajuste
